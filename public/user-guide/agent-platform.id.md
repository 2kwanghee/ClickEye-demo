# Platform Agent

ClickEye mendukung berbagai platform AI agent. Pilihlah platform yang sesuai dengan karakteristik proyek Anda.

## Platform yang Didukung

### Claude Code
Agent kode berbasis model Claude dari Anthropic.

**Karakteristik**:
- Pembuatan dan peninjauan kode berkualitas tinggi
- Pemahaman konteks multi-berkas
- Mendukung MCP (Model Context Protocol)

**Memulai**:
```bash
# Instal Claude Code
npm install -g @anthropic-ai/claude-code

# Jalankan dari direktori proyek Anda
claude
```

### Gemini CLI
Agent baris perintah berbasis model Gemini dari Google.

**Karakteristik**:
- Jendela konteks berkapasitas besar
- Integrasi dengan layanan Google

### Cursor
Platform editor kode berbasis AI.

**Karakteristik**:
- Lingkungan pengembangan terpadu
- Asisten AI secara real-time

### Codex
Agent berbasis Codex dari OpenAI.

**Karakteristik**:
- Mendukung beragam model
- Konfigurasi yang dapat diperluas
