# エージェントプラットフォーム

ClickEye はさまざまな AI エージェントプラットフォームに対応しています。プロジェクトの特性に合ったプラットフォームを選択してください。

## 対応プラットフォーム

### Claude Code
Anthropic の Claude モデルをベースにしたコードエージェントです。

**特徴**:
- 高品質なコード生成およびレビュー
- マルチファイルのコンテキスト理解
- MCP（Model Context Protocol）対応

**始め方**:
```bash
# Claude Code のインストール
npm install -g @anthropic-ai/claude-code

# プロジェクトディレクトリで実行
claude
```

### Gemini CLI
Google の Gemini モデルをベースにしたコマンドラインエージェントです。

**特徴**:
- 大容量のコンテキストウィンドウ
- Google サービスとの統合

### Cursor
AI ベースのコードエディタプラットフォームです。

**特徴**:
- 統合開発環境
- リアルタイム AI アシスタント

### Codex
OpenAI の Codex をベースにしたエージェントです。

**特徴**:
- さまざまなモデルに対応
- 拡張可能な設定
