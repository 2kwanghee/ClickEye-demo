# 에이전트 플랫폼

ClickEye는 다양한 AI 에이전트 플랫폼을 지원합니다. 프로젝트 특성에 맞는 플랫폼을 선택하세요.

## 지원 플랫폼

### Claude Code
Anthropic의 Claude 모델 기반 코드 에이전트입니다.

**특징**:
- 고품질 코드 생성 및 리뷰
- 멀티파일 컨텍스트 이해
- MCP(Model Context Protocol) 지원

**시작하기**:
```bash
# Claude Code 설치
npm install -g @anthropic-ai/claude-code

# 프로젝트 디렉토리에서 실행
claude
```

### Gemini CLI
Google의 Gemini 모델 기반 커맨드라인 에이전트입니다.

**특징**:
- 대용량 컨텍스트 윈도우
- Google 서비스 통합

### Cursor
AI 기반 코드 편집기 플랫폼입니다.

**특징**:
- 통합 개발 환경
- 실시간 AI 어시스턴트

### Codex
OpenAI의 Codex 기반 에이전트입니다.

**특징**:
- 다양한 모델 지원
- 확장 가능한 설정

> 상세 내용은 추후 업데이트 예정입니다 (24S-186).
