# AI Team 관리

AI Team 대시보드는 프로젝트의 AI 에이전트 팀을 오케스트레이션하는 3계층 시스템입니다.

---

## 대시보드 접근 방법

1. ClickEye 웹 → 프로젝트 목록에서 프로젝트 선택
2. 프로젝트 상세 페이지 → 좌측 사이드바 또는 상단 탭에서 **AI Team** 클릭
3. URL: `/projects/{projectId}/ai-team`

---

## 3계층 구조 이해

| 계층 | 담당자 | 역할 |
|------|-------|------|
| **1계층 — 사람** | 사용자 | 작업 요청, 단계 승인, 최종 결정 |
| **2계층 — PM AI** | AI | 작업 분해, 초안 생성, 리뷰 조율, 10단계 파이프라인 실행 |
| **3계층 — AI Team** | AI 에이전트들 | 서브태스크 병렬 실행 |

---

## 첫 작업 요청하기

세션이 없는 경우 화면 중앙에 **첫 작업 요청하기** 버튼이 표시됩니다.

1. **첫 작업 요청하기** (또는 헤더의 **새 작업 요청**) 클릭
2. 모달에서 작업 제목과 설명 입력
3. **생성** 클릭 → 세션이 `requested` 상태로 생성됩니다

### 여러 세션 관리

상단 탭에서 세션별로 전환할 수 있습니다. 각 탭에는 세션 제목과 현재 단계가 표시됩니다.

---

## 단계(Phase) 흐름

| 단계 | 설명 | 사용자 액션 |
|------|------|-----------|
| requested | 작업 요청됨 | — |
| decomposed | 서브태스크 분해됨 | — |
| assigned | PM에게 배정됨 | **AI 초안 생성** 버튼 클릭 |
| drafting | AI 초안 작성 중 | — |
| reviewing | 리뷰 중 | — |
| integrating | 통합 중 | — |
| validating | 검증 중 | **승인** 버튼 클릭 |
| approved | 승인됨 | — |
| transitioning | 전환 중 | — |
| completed | 완료 | — |

---

## AI 초안 생성

세션 단계가 `assigned`가 되면 **AI 초안 생성** 버튼이 활성화됩니다.

1. **AI 초안 생성** 버튼 클릭
2. AI가 순서대로 두 가지 작업을 자동으로 실행합니다:
   - ① 서브태스크 초안 생성 (작업을 작은 단위로 분해)
   - ② Linear 이슈 자동 등록 (Linear 자격증명이 저장된 경우)

### AI 초안 생성 후 화면

- **리뷰 라운드**: AI가 작성한 초안이 표시됩니다
- **Linear 동기화 힌트**: 등록된 Linear 이슈 목록이 표시됩니다

---

## Linear 동기화 힌트

AI 초안 생성이 성공하면 "Linear 동기화" 패널이 나타납니다.

### 이슈 자동 등록 성공 시

```
✓ Linear 이슈 생성 완료: 24S-123, 24S-124, 24S-125
```

등록된 이슈 ID(예: `24S-123`)를 Linear에서 확인할 수 있습니다.

### Linear 자격증명 미저장 시

```
Linear 자격증명이 없습니다. 설정에서 API 키를 저장하세요 →
```

이 경우 [Linear 연동 설정 가이드](/guide/linear-integration-setup)를 참고하여 API 키를 먼저 저장하세요.

---

## 역할 할당 (서브태스크 확인)

AI Team 섹션(3계층)에서 각 서브태스크 카드를 확인합니다.

각 서브태스크 카드에는 다음 정보가 표시됩니다:

- **역할**: 담당 에이전트 역할 (예: backend, frontend, devops)
- **제목**: 서브태스크 이름
- **초안 요약**: AI가 작성한 작업 개요

서브태스크는 30초마다 자동으로 갱신됩니다.

---

## 최종 승인

PM AI가 작업을 완료하고 검증(`validating`) 단계에 도달하면 **승인** 버튼이 활성화됩니다.

1. 리뷰 라운드의 AI 초안 내용을 확인합니다.
2. 이상이 없으면 **승인** 버튼을 클릭합니다.
3. 세션이 `approved` → `completed` 순으로 전환됩니다.

---

## 리스크 플래그

세션에 잠재적 문제가 감지되면 상단에 리스크 플래그가 표시됩니다.

| 플래그 | 의미 |
|-------|------|
| 복잡도 높음 | 작업 범위가 너무 광범위함 |
| 의존성 충돌 | 서브태스크 간 의존 관계 문제 |
| 키 미설정 | 필수 API 키가 없음 |

---

## 새로고침 및 실시간 업데이트

- **새로고침** 버튼: 세션 목록과 현재 세션 요약을 수동으로 갱신합니다.
- **AI Team 섹션**: 30초마다 자동으로 갱신됩니다.

---

## 자주 묻는 질문

**Q. AI 초안 생성을 여러 번 클릭하면 어떻게 되나요?**
A. 이미 초안이 있는 경우 덮어쓰지 않습니다. `assigned` 단계에서만 버튼이 활성화됩니다.

**Q. Linear 이슈 등록에 실패했는데 다시 시도하려면?**
A. [설정 페이지](/settings/linear)에서 API 키를 저장한 뒤 AI Team 페이지를 새로고침하세요. AI 초안 생성 시 이슈 등록이 재시도됩니다.

**Q. 세션을 삭제할 수 있나요?**
A. 현재 UI에서는 세션 삭제 기능이 제공되지 않습니다. 새 세션은 언제든지 추가할 수 있습니다.
