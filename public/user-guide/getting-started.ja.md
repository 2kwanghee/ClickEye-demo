# はじめに

ClickEye へようこそ。

## 概要

ClickEye は、AI エージェントによる開発自動化ソリューションを構築するためのプラットフォームです。ブラウザ上の 7-Step ウィザードでソリューションを設計し、ZIP としてダウンロードすると、ローカル環境で AI エージェントが開発を自動化します。

## プラットフォーム構成

- **Web ダッシュボード**: ソリューション設計およびプロジェクト管理
- **エージェントプラットフォーム**: Claude Code、Gemini CLI、Cursor、Codex に対応
- **CLI ツール**: パワーユーザー向けのコマンドラインインターフェース

## クイックスタート

1. ダッシュボードで **新しいソリューション** ボタンをクリックします。
2. 7-Step ウィザードに沿ってプロジェクトを構成します。
3. ZIP をダウンロードし、ローカル環境で解凍します。
4. 選択した AI エージェントプラットフォームを起動します。
