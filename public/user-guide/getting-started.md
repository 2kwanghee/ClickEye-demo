# 시작하기

ClickEye에 오신 것을 환영합니다.

## 개요

ClickEye는 AI 에이전트 개발 자동화 솔루션 빌더 플랫폼입니다. 브라우저에서 7-Step 위저드로 솔루션을 설계하고 ZIP으로 다운로드하면, 로컬 환경에서 AI 에이전트가 개발을 자동화합니다.

## 플랫폼 구성

- **웹 대시보드**: 솔루션 설계 및 프로젝트 관리
- **에이전트 플랫폼**: Claude Code, Gemini CLI, Cursor, Codex 지원
- **CLI 도구**: 파워유저를 위한 커맨드라인 인터페이스

## 빠른 시작

1. 대시보드에서 **새 솔루션** 버튼을 클릭합니다.
2. 7-Step 위저드를 따라 프로젝트를 구성합니다.
3. ZIP을 다운로드하여 로컬 환경에서 압축을 해제합니다.
4. 선택한 AI 에이전트 플랫폼을 실행합니다.
