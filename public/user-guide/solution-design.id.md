# Perancangan Solusi

## Wizard 7-Step

Rancang solusi pengembangan AI Anda melalui wizard 7-Step, fitur inti ClickEye. Setelah menyelesaikan setiap langkah, Anda dapat mengunduhnya sebagai berkas ZIP.

## Panduan per Langkah

### Step 1 — Informasi Dasar Proyek
Masukkan nama, deskripsi, dan tujuan proyek.

### Step 2 — Pemilihan Platform
Pilih platform AI agent yang akan Anda gunakan.
- Pilih salah satu di antara Claude Code, Gemini CLI, Cursor, dan Codex

### Step 3 — Pemilihan Stack
Pilih bahasa pemrograman dan framework.

### Step 4 — Konfigurasi Agent
Pilih agent yang akan digunakan dalam proyek.

### Step 5 — Konfigurasi Skill
Pilih modul skill yang Anda butuhkan.

### Step 6 — Konfigurasi MCP
Atur konfigurasi server MCP (Model Context Protocol).

### Step 7 — Tinjau dan Hasilkan
Tinjau seluruh konfigurasi, lalu unduh ZIP.

## Struktur ZIP

Saat Anda mengekstrak ZIP yang telah diunduh, struktur berikut akan terbentuk:

```
my-project/
├── .claude/          # Pengaturan Claude Code
├── .env.example      # Templat variabel lingkungan
├── CLAUDE.md         # Panduan proyek
└── README.md         # Panduan memulai
```

> Detail lebih lanjut akan diperbarui kemudian (24S-186).
