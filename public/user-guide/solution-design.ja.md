# ソリューション設計

## 7-Step ウィザード

ClickEye の中核機能である 7-Step ウィザードを通じて、AI 開発ソリューションを設計します。各ステップを完了すると、ZIP ファイルとしてダウンロードできます。

## ステップ別ガイド

### Step 1 — プロジェクトの基本情報
プロジェクト名、説明、目的を入力します。

### Step 2 — プラットフォーム選択
使用する AI エージェントプラットフォームを選択します。
- Claude Code、Gemini CLI、Cursor、Codex から選択

### Step 3 — スタック選択
プログラミング言語およびフレームワークを選択します。

### Step 4 — エージェント構成
プロジェクトで使用するエージェントを選択します。

### Step 5 — スキル構成
必要なスキルモジュールを選択します。

### Step 6 — MCP 構成
MCP（Model Context Protocol）サーバーの設定を構成します。

### Step 7 — レビューと生成
すべての設定をレビューした上で ZIP をダウンロードします。

## ZIP の構造

ダウンロードした ZIP を解凍すると、次の構造が生成されます:

```
my-project/
├── .claude/          # Claude Code 設定
├── .env.example      # 環境変数テンプレート
├── CLAUDE.md         # プロジェクトガイド
└── README.md         # スタートガイド
```
