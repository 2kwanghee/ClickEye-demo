# 솔루션 설계

## 7-Step 위저드

ClickEye의 핵심 기능인 7-Step 위저드를 통해 AI 개발 솔루션을 설계합니다. 각 단계를 완료하면 ZIP 파일로 다운로드할 수 있습니다.

## 단계별 안내

### Step 1 — 프로젝트 기본 정보
프로젝트 이름, 설명, 목적을 입력합니다.

### Step 2 — 플랫폼 선택
사용할 AI 에이전트 플랫폼을 선택합니다.
- Claude Code, Gemini CLI, Cursor, Codex 중 선택

### Step 3 — 스택 선택
프로그래밍 언어 및 프레임워크를 선택합니다.

### Step 4 — 에이전트 구성
프로젝트에 사용할 에이전트를 선택합니다.

### Step 5 — 스킬 구성
필요한 스킬 모듈을 선택합니다.

### Step 6 — MCP 구성
MCP(Model Context Protocol) 서버 설정을 구성합니다.

### Step 7 — 검토 및 생성
전체 설정을 검토한 뒤 ZIP을 다운로드합니다.

## ZIP 구조

다운로드한 ZIP을 압축 해제하면 다음 구조가 생성됩니다:

```
my-project/
├── .claude/          # Claude Code 설정
├── .env.example      # 환경변수 템플릿
├── CLAUDE.md         # 프로젝트 가이드
└── README.md         # 시작 가이드
```

> 상세 내용은 추후 업데이트 예정입니다 (24S-186).
